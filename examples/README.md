# Examples

This directory contains user-visible TranslationStack demo projects.

## Available Demos

- [Building Effective Agents](building-effective-agents/) - a complete English-to-Simplified-Chinese TranslationStack project with source material, chunk manifest, glossary, style guide, translation chunks, AI review issues, revision history, run records, and exported output.

Each demo is ordinary files on disk. You can inspect it directly in GitHub, run the validator against it, or copy it into `.translationstack/<project-id>/` as a starting point for local experimentation.
